## Mission

**17 veterans die by suicide every day.**

When a veteran is in crisis, they don't call a hotline‚Äîthey text their battle buddy. But that buddy has no tools and no backup.

**Forward Operating Base gives the helping veteran instant access to local resources, evidence-based responses, and warm handoff to peer counselors.**

We're not building another app for veterans in crisis. **We're building tools for veterans who help other veterans.**

## Flagship: Buddy Backup Command

The Battle Buddy System is the core innovation:

- **Peer Support Rapid Connect** - Instant resource access for helping veterans
- **Evidence-Based Responses** - Scripted guidance for crisis situations
- **Local Resource Mapping** - Real-time connection to VA, VSOs, community resources
- **Warm Handoff** - Seamless transition to professional peer counselors
- **No Wrong Door** - Every interaction leads to help, not dead ends
